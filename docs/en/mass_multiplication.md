# Mass Multiplication

## Grasshopper

- Component Name:
    - `Mass Multiplication`
- Input:
    - l: List of values.
- Output:
    - R: All values in the input list multiplied.
    - Pr (`List`): Partial results (mass multiplication result up to nth value).

## Difficulty

### Easy

- There are only three values in the list.
- All values in the list satisfy `-10 <= v <= 10`.
