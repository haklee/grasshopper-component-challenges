# Larger Than

## Grasshopper

- Component Name:
    - `Larger Than`
- Input:
    - A: First item.
    - B: Second item.
- Output:
    - `>` (`boolean`): A is greater than B.
    - `>=` (`boolean`): A is greater or equal to B.

## Difficulty

### Easy

- [.ghx file](../../problems/larger_than_easy.ghx)
- conditions:
    - Both input A and B are `float` type.
