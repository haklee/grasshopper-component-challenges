# Bounds

## Grasshopper

- Component Name:
    - `Bounds`
- Input:
    - N: List of numbers.
- Output:
    - I (`Interval`): Numeric domain defined with minimum and maximum values in the input list.

## Difficulty

### Easy

- [.ghx file](../../problems/bounds_easy.ghx)
- conditions:
    - There are only five values in the list.
    - All values in the list satisfies `-10 <= v <= 10`.
