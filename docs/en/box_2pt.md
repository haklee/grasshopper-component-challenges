# Box 2Pt

## Grasshopper

- Component Name:
    - `Box 2Pt`
- Input:
    - A (`Point3d`): First corner.
    - B (`Point3d`): Second corner.
    - P (`Plane`): Base plane.
- Output:
    - B (`Box`): Box created with two input corners `A` and `B`

## Difficulty

### Standard

- [.ghx file](../../problems/box_2pt_easy.ghx)
- conditions:
    - Plane input is not given and default plane input(World XY) is used.
